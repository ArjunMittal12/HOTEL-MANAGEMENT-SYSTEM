# HOTEL-MANAGEMENT-SYSTEM

A lightweight JavaFX-based Hotel Management System (mini project).

This repository contains a small desktop app that demonstrates a layered Java application with a JavaFX UI, a service layer, simple file-based persistence, and modular Java (module-info). The primary working application lives in the `mini-project/` folder.

Why this repo is worth contributing to
- A compact, readable codebase (Java + JavaFX) suitable for beginners and intermediate contributors.
- Clear separation of concerns (controllers, service, models, storage) makes it easy to pick focused improvements (tests, logging, persistence hardening).
- Good first issues can be added (tests, UI polish, UX fixes, documentation, CI) and accepted quickly.

Quick links
- Project code: `mini-project/`
- Architecture notes: `mini-project/APPLICATION_ARCHITECTURE.md` and `mini-project/APPLICATION_ARCHITECTURE_VIVA_NOTES.md`
- Design and run details: `mini-project/ARCHITECTURE_AND_FUNCTIONALITY.md`

Want to contribute?
1. Fork the repository on GitHub.
2. Create a branch for your change: `git checkout -b feat/short-description`.
3. Make small, focused changes with clear commit messages.
4. Run the app and tests locally (see `mini-project/README.md`).
5. Open a Pull Request with a short description of your change and any screenshots or logs.

Maintainer tips to attract contributors
- Add a few `good first issue` labeled issues that are small and self-contained (fix a bug, add a unit test, improve a UI label).
- Provide a `CONTRIBUTING.md` that lists the local setup and coding style.
- Include screenshots and a short demo GIF in the README so visitors quickly understand the project.
